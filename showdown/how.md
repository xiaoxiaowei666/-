2.初始化:
  2.1游戏开始收到的消息格式
  ```json
{
"type": "notify",
"stage": "beginning",
"handCards": ['S2', 'H2', 'H2', 'C2', 'D2', 'S3', 'H3', 'D3', 'S4', 
'D5', 'C6', 'H7', 'C7', 'C8', 'D8', 'DT', 'HJ', 'DJ', 'HQ', 'HQ', 'CQ', 
'SK', 'H9', 'H9', 'C9', 'D9', 'HR'],
"myPos": 1,
}
```
开始渲染：看看消息格式中的type：notify且为stage:beginning且myPos为当前客户端的client_index值的话就去依据handCards中的元素进行渲染去,例如:S2 就可以去:./images/cards/S2.jpg 进行渲染即可!------------我自己的手牌，能够看到！
然后因为总共是108张牌，四个人:每个人初始为27张牌,除了自己可以给其余的人都记录27张牌的数量，然后再根据牌的数量去渲染相应数量的:./images/cards/back.bmp 作为被遮住的牌的！
然后再根据自己的myPos 把相应的玩家位置 进行安排好，我的上家为：(myPos-1)%4+4,我的下家为：(mypos+1)%4 这种！然后再把剩余的一家进行渲染即可！-------三个人机的手牌！被遮住了无法看到！
